PLZBUM IMDB
===========

Fetches the movie title and prints it to the channel for any imdb link posted in the format https://www.imdb.com/title/(.+)/

Configuration
-------------

Create an API key for OMDB here https://www.omdbapi.com/apikey.aspx

Update the `~/.sopel/default.cfg` to include

```
[plzbum_imdb]
api_key = XYZ
```
